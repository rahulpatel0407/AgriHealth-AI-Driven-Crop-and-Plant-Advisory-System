# Privacy Policy (Draft)

## Consent
We collect leaf images only after explicit consent. Users can opt out and still receive generic advice.

## Data retention
- Images: stored for 90 days by default for improving model quality.
- Metadata: retained for 1 year for analytics and support.

## PII handling
- Phone numbers are encrypted at rest.
- No raw PII is logged to public logs.

## Deletion
Users can request deletion of images and metadata via support. Deletion completes within 30 days.
